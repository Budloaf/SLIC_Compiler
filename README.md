# SLIC_Compiler
This is a copy of my project from Compiler Construction class at University.

## Background
This project compiles SLIC code to GSTAL using C, Flex, and Bison. If you do not know what SLIC or GSTAL are, that is because SLIC is a language my professor, Dr. Bryan Crawley, created for educational purposes (the class mentioned above). GSTAL is a pseudo-assembly language created for a similar purpose and derived from STAL, as explained in the GSTAL specifications.
